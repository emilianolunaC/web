El mundo de las Ecuaciones Diferenciales

Aprende a describir cómo evoluciona el mundo. Desde el crecimiento poblacional hasta la transferencia de calor, todo se rige por tasas de cambio.

Ejemplo fundamental:

$$\frac{dy}{dx} = ky \implies y = Ce^{kx}$$

Fundamentales en física (vibraciones y resortes). Incluyen hasta la segunda derivada.

Involucran múltiples variables independientes y derivadas parciales.

Guía para Resolver Ecuaciones Diferenciales

🔹 1er Orden

Variables Separables:

$$\frac{dy}{dx} = g(x)h(y) \implies \int \frac{1}{h(y)} dy = \int g(x) dx$$

Ecuación Lineal:

Factor integrante:

$$\mu(x) = e^{\int P(x)dx}$$

Solución:

$$\mu(x)y = \int \mu(x)Q(x)dx$$

🔹 2do Orden Homogénea

Forma: \(ay'' + by' + cy = 0\)

Usar ecuación característica: \(ar^2 + br + c = 0\)

  • ✅ Reales diferentes:
    \(y = C_1e^{r_1x} + C_2e^{r_2x}\)
  • ✅ Reales iguales:
    \(y = (C_1 + C_2x)e^{rx}\)
  • ✅ Complejas:
    \(y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)\)

🔹 Parciales (EDP)

Separación de variables:

Proponer: \(u(x,t) = X(x)T(t)\)

$$\frac{1}{T}\frac{dT}{dt} = \frac{1}{X}\frac{d^2X}{dx^2} = -\lambda$$

🔥 Resumen: 1er orden (separar), 2do orden (característica), Parciales (separación).
